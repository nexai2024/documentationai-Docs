## Overview

NexAI uses advanced machine learning to scan incoming messages, emails, and websites for suspicious patterns.

## Detection Process

1. Analyze text for common scam keywords and phrases
2. Check sender reputation and domain history
3. Cross-reference against real-time scam databases
4. Assign a risk score to each message

If a risk score exceeds your configured threshold, NexAI issues an instant alert to help you stop the scam before it starts.
